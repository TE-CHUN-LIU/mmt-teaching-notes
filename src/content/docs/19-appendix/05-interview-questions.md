這份是用 22 種特質設計的面談題庫。面談時依「你想驗證的特質」挑題：先問情境提問，再用追問話術與深入追問探，最後對照答案判讀。

---

## 0 創意家（創新）

> 你天生不安於現狀，總想找到新的解法或是顛覆性的點子，對未知充滿興奮感與渴望。

**觀察重點**：是否不安於現狀、喜歡找新解法、能不能接受未知與試錯。

| 階段 | 內容 |
|------|------|
| 情境提問 | 如果公司有一套已經運作很久、也還算穩定的流程，但你發現它其實可以更有效率，你會怎麼做？ |
| 追問話術 | 你會直接提出新方法，還是先照原本的流程做？為什麼？ |
| 深入追問 | 如果主管說：「以前都這樣做，先不要改。」你會怎麼反應？ |
| 答案判讀 | 若對方會主動提出優化，代表創新與改善意識強；若會先觀察再提，代表創新但懂得拿捏時機；若完全不想改，可能較重視穩定，不一定適合高度變動型角色。 |

## 1 整合者（資訊收集者）

> 你對資訊的敏感度極高，喜歡探索各種領域的知識，像一塊吸水海綿，不斷擴充自己的資料庫。

**觀察重點**：是否喜歡蒐集資料、研究、比較、建立知識庫。

| 階段 | 內容 |
|------|------|
| 情境提問 | 如果你接到一個從沒做過的新任務，時間有限，你會先做什麼？ |
| 追問話術 | 你會先開始做，還是先查資料、問人、整理資訊？為什麼？ |
| 深入追問 | 你通常會查到什麼程度才覺得「可以開始行動了」？ |
| 答案判讀 | 若對方會主動蒐集資訊、比較案例，代表學習與資料整合力強；若查太久才行動，可能有資訊過載或拖延傾向；若完全不查就做，可能執行快但風險判斷不足。 |

## 2 敏銳者（敏銳的直覺）

> 你的第六感特別發達，能夠感受到環境與人心的細微變化，總是在對的時間做出對的決策。

**觀察重點**：是否能感受到人與環境的細微變化，並用直覺輔助判斷。

| 階段 | 內容 |
|------|------|
| 情境提問 | 如果你在會議中感覺某位同事雖然嘴上說沒問題，但狀態明顯怪怪的，你會怎麼做？ |
| 追問話術 | 你會相信自己的感覺去關心他，還是等他主動說？為什麼？ |
| 深入追問 | 你過去有沒有曾經「感覺哪裡不對」，後來證明是真的經驗？ |
| 答案判讀 | 若能描述細節與後續行動，代表直覺敏銳且有觀察力；若只憑感覺但無法驗證，可能容易主觀投射；若完全不留意氛圍，可能較偏任務導向。 |

## 3 美學家（高層級魅力與吸引力）

> 你不需要刻意經營形象，天生就散發吸引人的氛圍，讓人忍不住想靠近你、欣賞你。

**觀察重點**：是否自然吸引他人、是否具備存在感與被關注的能力。

| 階段 | 內容 |
|------|------|
| 情境提問 | 如果你進入一個完全陌生的聚會或場合，你通常會怎麼讓自己融入？ |
| 追問話術 | 你會主動認識人，還是等別人來找你？為什麼？ |
| 深入追問 | 你覺得別人通常是因為什麼而想靠近你？ |
| 答案判讀 | 若對方自然能成為焦點，且不過度刻意，代表魅力與吸引力明顯；若過度依賴外在包裝，可能形象意識強但內在穩定度待觀察；若抗拒被看見，可能魅力沒有被開發。 |

## 4 挑戰者（爭強好勝的企圖心）

> 你不僅渴望成功，更希望獲得他人的認可與讚賞。你的競爭精神讓你不斷挑戰更高的目標。

**觀察重點**：是否渴望成就、排名、被肯定，面對競爭是否會被激發。

| 階段 | 內容 |
|------|------|
| 情境提問 | 如果團隊裡有人業績或表現比你更好，你通常會怎麼看待這件事？ |
| 追問話術 | 你會覺得被刺激、想超越他，還是覺得各做各的就好？為什麼？ |
| 深入追問 | 當你努力很久卻沒有被看見時，你會怎麼調整？ |
| 答案判讀 | 若被競爭激發，代表企圖心強；若需要外在掌聲才有動力，可能高度依賴認可；若排斥競爭，可能較適合穩定協作型角色。 |

## 5 傳教者（價值觀強烈）

> 你有明確的信念與價值體系，重視穩定與安全，願意花時間與他人分享你的理念與經驗，成為價值觀的傳遞者。

**觀察重點**：是否有明確信念、原則感強、重視穩定與安全。

| 階段 | 內容 |
|------|------|
| 情境提問 | 如果公司要求你用一種你不太認同的方式達成目標，但它是有效的，你會怎麼做？ |
| 追問話術 | 你會先配合，還是會提出你的顧慮？為什麼？ |
| 深入追問 | 你有沒有什麼事情是「就算有利益，你也不會做」？ |
| 答案判讀 | 若能清楚說出底線，代表價值觀穩定；若過度堅持己見，可能彈性不足；若完全沒有原則，可能短期配合高，但長期信任感不足。 |

## 6 演繹家（人緣與社交優勢）

> 你擁有絕佳的語言能力和社交魅力，總能在人群中迅速找到共鳴，建立連結，成為受歡迎的焦點人物。

**觀察重點**：是否能快速與人建立連結、擅長溝通與社交互動。

| 階段 | 內容 |
|------|------|
| 情境提問 | 如果你被安排參加一場都是陌生人的商務聚會，你會怎麼開始建立關係？ |
| 追問話術 | 你會先找什麼類型的人聊天？為什麼？ |
| 深入追問 | 你通常怎麼讓對方願意繼續跟你保持聯絡？ |
| 答案判讀 | 若能主動破冰並建立後續連結，代表社交優勢強；若只會聊天但無法延續關係，可能偏表層社交；若容易緊張退縮，可能社交能量較低。 |

## 7 守護者（重視家庭與歸屬感）

> 你對家庭觀念根深蒂固，認為幸福與滿足來自穩固的情感關係與安全的生活環境。

**觀察重點**：是否重視安全感、關係穩定、情感連結與歸屬。

| 階段 | 內容 |
|------|------|
| 情境提問 | 如果一份工作收入更高，但需要長期犧牲陪伴家人或重要關係的時間，你會怎麼選？ |
| 追問話術 | 你會優先考慮收入成長，還是生活與關係的穩定？為什麼？ |
| 深入追問 | 你覺得「成功」裡面，家庭或重要關係佔多少比例？ |
| 答案判讀 | 若重視關係穩定，代表歸屬感需求高；若能平衡事業與家庭，成熟度較高；若完全被家庭牽制，可能在高壓拓展型角色中受限。 |

## 8 啟導者（內斂的企圖心）

> 與渴望黃袍加身不同，你的野心藏得更深，追求的是穩固的地位與實力，而非表面的風光。

**觀察重點**：是否有野心但不張揚，重視實力、地位、長期累積。

| 階段 | 內容 |
|------|------|
| 情境提問 | 如果你有機會負責一個重要專案，但不一定會被大家看見，你還會想接嗎？ |
| 追問話術 | 你在意的是舞台曝光，還是實際掌握權與成果？為什麼？ |
| 深入追問 | 你比較想成為被看見的明星，還是背後真正有影響力的人？ |
| 答案判讀 | 若重視實權與長期累積，代表內斂企圖心強；若完全不想被看見，可能自我推進力不足；若只要掌聲不要責任，則偏外顯成就需求。 |

## 9 穿透者（直指問題核心）

> 你善於快速辨別問題的關鍵所在，並直接切入核心提出解決方案，展現高效與務實的一面。

**觀察重點**：是否能快速抓重點、切入問題本質、務實解決。

| 階段 | 內容 |
|------|------|
| 情境提問 | 如果團隊開會討論很久，但大家一直繞圈子，沒有結論，你會怎麼做？ |
| 追問話術 | 你會直接點出問題，還是先觀察氣氛？為什麼？ |
| 深入追問 | 如果你講得太直接，讓人不舒服，你會怎麼處理？ |
| 答案判讀 | 若能快速整理重點並推進，代表問題解決力強；若過於直接，可能人際敏感度不足；若不敢切入，可能判斷有但表達力不足。 |

## 10 顯化者（幸運與資源磁場）

> 你像一個資源吸引器，總能找到貴人與機會，自帶成功與擴展的能量。

**觀察重點**：是否容易吸引資源、善用人脈、看見機會。

| 階段 | 內容 |
|------|------|
| 情境提問 | 如果你現在要完成一個目標，但手上資源不足，你通常會怎麼開始？ |
| 追問話術 | 你會自己硬做，還是去找人、找資源、找機會？為什麼？ |
| 深入追問 | 你過去有沒有遇過貴人或資源剛好出現的經驗？你做了什麼讓它發生？ |
| 答案判讀 | 若會主動連結資源，代表資源磁場與機會感強；若只等待別人幫忙，可能依賴運氣；若完全靠自己，可能獨立但槓桿能力不足。 |

## 11 權威者（得體與優雅）

> 你天生注重外在形象與內在氣質，懂得如何在各種場合保持自信與魅力，給人留下深刻的印象。

**觀察重點**：是否重視形象、禮儀、場合感與表達分寸。

| 階段 | 內容 |
|------|------|
| 情境提問 | 如果你要代表公司出席一場重要場合，你會怎麼準備自己？ |
| 追問話術 | 你會特別注意穿著、談吐、禮儀嗎？為什麼？ |
| 深入追問 | 你覺得一個人的外在形象會不會影響別人對他的信任？ |
| 答案判讀 | 若能根據場合調整形象，代表得體感佳；若過度在意外表，可能有形象焦慮；若完全不重視場合感，可能在高端客戶或商務場合吃虧。 |

## 12 趨勢家（利他與成就他人）

> 你喜歡讓他人站上舞台，樂於為別人創造機會與條件，幫助他們實現夢想與價值。

**觀察重點**：是否願意支持他人、讓別人成功、具備教練或助人特質。

| 階段 | 內容 |
|------|------|
| 情境提問 | 如果你協助一位夥伴成功，但最後聚光燈都在他身上，你會怎麼感受？ |
| 追問話術 | 你會覺得開心，還是覺得自己的功勞沒有被看見？為什麼？ |
| 深入追問 | 你過去有沒有主動幫助別人成長或成功的經驗？ |
| 答案判讀 | 若真心因他人成就而滿足，代表利他特質強；若期待被感謝與認可，代表助人中仍有自我價值需求；若不願意成就他人，可能較偏個人成就導向。 |

## 13 改革者（挑戰階級與制度）

> 你擁有革命性的視野，擅長打破框架、挑戰權威，推動社會或組織變革，是改革與創新的先鋒。

**觀察重點**：是否敢挑戰權威、打破框架、推動改革。

| 階段 | 內容 |
|------|------|
| 情境提問 | 如果你發現公司某個制度明顯不合理，但大家都習慣了，你會怎麼做？ |
| 追問話術 | 你會選擇忍耐、私下抱怨，還是提出改革建議？為什麼？ |
| 深入追問 | 如果你的建議會挑戰到主管或既得利益者，你還會說嗎？ |
| 答案判讀 | 若能理性挑戰制度，代表改革能量強；若只想推翻但沒有方案，可能反骨大於建設；若完全服從不合理制度，可能較不適合創新變革角色。 |

## 14 使命家（懂人心的啟蒙者）

> 你對人性有敏銳的洞察力，擅長用輕鬆說故事的方式，自然的啟發他人，讓人感受到被理解與支持。

**觀察重點**：是否能理解他人、用故事或引導方式啟發對方。

| 階段 | 內容 |
|------|------|
| 情境提問 | 如果朋友或同事陷入低潮，一直想不開，你通常會怎麼陪他？ |
| 追問話術 | 你會直接給建議，還是先聽他說？為什麼？ |
| 深入追問 | 你會用什麼方式讓對方自己想通，而不是只聽你的答案？ |
| 答案判讀 | 若能先理解再引導，代表懂人心與啟蒙能力強；若急著說教，可能洞察有但陪伴感不足；若只共感但無法推進，可能溫暖但缺乏引導力。 |

## 15 策略家（策略力與多謀善斷）

> 你思路縝密、計劃周全，擅長分析情勢與設計路徑，讓複雜的問題變得清晰可解。

**觀察重點**：是否能分析局勢、規劃路徑、做出取捨。

| 階段 | 內容 |
|------|------|
| 情境提問 | 如果你要在三個月內完成一個重要目標，但資源有限，你會怎麼規劃？ |
| 追問話術 | 你會先做哪些事情？哪些事情會先放掉？為什麼？ |
| 深入追問 | 如果中途情勢改變，你會怎麼調整策略？ |
| 答案判讀 | 若能清楚拆解目標、資源、路徑與取捨，代表策略力強；若想很多但不行動，可能過度謀劃；若只憑感覺衝，可能執行快但策略不足。 |

## 16 執行家（倔強與自尊心強）

> 你內心有一份驕傲，追求實際成果來證明自己的價值，比起安慰性的言語，你更希望得到實際的認同與尊重。

**觀察重點**：是否重視實際成果、尊嚴、被尊重，不喜歡被敷衍安慰。

| 階段 | 內容 |
|------|------|
| 情境提問 | 如果你很努力完成一件事，但主管只隨口說「辛苦了」，沒有實際肯定或回饋，你會怎麼想？ |
| 追問話術 | 你會覺得被鼓勵，還是覺得不夠被尊重？為什麼？ |
| 深入追問 | 當別人質疑你的能力時，你通常會怎麼反應？ |
| 答案判讀 | 若會用成果證明自己，代表自尊與成就驅動強；若容易因一句話受傷，可能自我保護強；若能接受回饋並修正，代表成熟度較高。 |

## 17 理想家（微社恐的矛盾感）

> 你渴望與人建立連結，但又需要保有自己的獨立空間，因此在人群中常表現出若即若離的態度。

**觀察重點**：是否想連結但又需要空間，社交上若即若離。

| 階段 | 內容 |
|------|------|

| 深入追問 | 活動結束後，你會覺得充電，還是覺得耗電？ |
| 答案判讀 | 若想社交但需要恢復時間，代表微社恐或內外向矛盾；若完全逃避人群，可能社交壓力高；若在人群中很自在，則社交能量較強。 |

## 18 感知者（莫名的恐懼）

> 你的敏感特質讓你容易感受到潛在的風險與威脅，即使現實中並不存在危險，也可能引發不安與焦慮感。

**觀察重點**：是否容易感知風險、焦慮、不安，甚至對尚未發生的事過度擔心。

| 階段 | 內容 |
|------|------|
| 情境提問 | 如果你要做一件新嘗試，但還沒有明確失敗跡象，只是心裡覺得不安，你會怎麼處理？ |
| 追問話術 | 你會先暫停觀察，還是邊做邊修正？為什麼？ |
| 深入追問 | 你通常怎麼分辨「直覺提醒」和「自己想太多」？ |
| 答案判讀 | 若能把恐懼轉成風險檢查，代表敏感度可被善用；若常被不安卡住，可能行動力受限；若完全忽略恐懼，可能冒進。 |

## 19 光之子（自我形象強烈）

> 你重視自我價值與形象，希望展現最好的自己，不僅讓人欣賞你的外在，更尊重你的內在信念與態度。

**觀察重點**：是否重視形象、身份感、價值呈現與他人眼中的自己。

| 階段 | 內容 |
|------|------|
| 情境提問 | 如果你要在公開場合介紹自己，你最希望別人記住你什麼？ |

## 20 計畫者（風險預測與計劃力）

> 你對細節極為敏感，總是習慣提前推演各種可能發生的情況，確保自己有萬全之策來應對變化。

**觀察重點**：是否會提前推演風險、準備備案、重視細節與安全。

| 階段 | 內容 |
|------|------|
| 情境提問 | 如果你要主辦一場重要活動，你會怎麼準備？ |
| 追問話術 | 你會先想流程，還是先想可能出錯的地方？為什麼？ |
| 深入追問 | 你通常會準備幾套備案？如果別人覺得你想太多，你會怎麼看？ |
| 答案判讀 | 若能提前預測風險並準備方案，代表計劃力強；若過度擔心細節，可能拖慢速度；若完全沒有備案，可能抗變能力不足。 |

## 21 建構者（穩定與秩序感）

> 你追求穩固的結構與規律，喜歡將事情按步驟推進，確保每一步都牢靠且有結果。

**觀察重點**：是否重視流程、規律、結構、步驟與可控感。

| 階段 | 內容 |
|------|------|
| 情境提問 | 如果你進入一個新團隊，發現大家做事很自由、沒有明確流程，你會怎麼適應？ |
| 追問話術 | 你會覺得彈性很舒服，還是會想建立規則？為什麼？ |
| 深入追問 | 當事情臨時變動、計畫被打亂時，你通常怎麼反應？ |
| 答案判讀 | 若會主動建立流程，代表秩序感與穩定性強；若遇到變動容易焦慮，可能彈性不足；若能在秩序與彈性間切換，成熟度高。 |
