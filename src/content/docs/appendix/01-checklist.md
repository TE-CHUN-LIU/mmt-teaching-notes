**前一天晚上**
- [ ] 投影片檔案在 USB ＋ 雲端各備一份
- [ ] 麥克風電池、講師包頭麥確認
- [ ] 場地空調溫度（怕冷的學員會分心）
- [ ] 學員名單列印、別針名牌
- [ ] 講義列印（每張 22 張牌的表格 ＋ 課程備課摘要前 3 頁）
- [ ] 報名 QR Code 海報（給課後想留資料的人）

**當天早上（8:00 到場）**
- [ ] 投影片開檔測試
- [ ] 試麥
- [ ] 試 wifi（QR Code 萬一連不上要有 backup）
- [ ] 板書 marker × 3 色（藍、紅、黑）
- [ ] 自備一瓶水

**講者個人 mindset**
- 第一堂 hook 一定要有 ── 沒有 hook，後面 11.5 小時都救不回來。
- 講錯不要硬撐 ── 學員問到不會的，誠實說「**這個我下午查一下、確定再給你答案**」── 比硬講出錯答案專業 10 倍。
- 不要怕停頓 ── **沉默 3 秒，大腦處理會更深**。
- 自己拿水喝、自己上廁所 ── 學員會跟著你的節奏。
