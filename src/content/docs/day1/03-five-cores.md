**核心訊息 ─ 五大核心是「層次遞進」的，不是並列的**

- 第一層 **四大能量** → 看你的「**底盤**」── 你的車是 SUV 還是跑車？
- 第二層 **天賦設計** → 看你的「**內裝配備**」── 車內有什麼系統？
- 第三層 **導師設計** → 看你的「**油門**」── 你被什麼驅動？
- 第四層 **陰影設計** → 看你的「**煞車 ／ 卡點**」── 你會被什麼絆住？
- 第五層 **年度策略** → 看你的「**導航**」── 當下該往哪邊開？

這個「車子比喻」對所有學員都很有效，要背起來。

**講者要強調的 mindset**

- 大部分人聽完只記得「天賦」── 他們只想知道「我有什麼超能力」。
- 你的工作是讓他們明白：**沒有陰影的天賦會傷人，沒有導師的天賦會卡死**。
- 標題「**如何把你的設計活得好**」── 重點是「**活得好**」，不是「活得快」「活得多」。MMT 講「適合度」，不講「成就高度」。

**互動點**

讓每位學員用一句話回答：「**如果你的生命可以重新設計一次，你最想保留什麼、最想拿掉什麼？**」── 收集到的答案，其實就會分布在「天賦 / 陰影」這兩格上。這個 setup 為下午的四大能量做鋪陳。

**FAQ 預先準備**
- Q：「五大核心要按順序看嗎？還是可以跳著看？」
  A：「**順序很重要**。沒看懂四大能量，去看天賦會誤判；沒看懂導師＋陰影，去談年度策略會空轉。」
- Q：「我可以只看天賦不看陰影嗎？」
  A：「可以，但你會錯過 50%。陰影是你『**已經支付但還沒拿回**』的能量。」
